# Dash Website

This repository contains the official website for the Dash Programming Language.

The site is built using React and TypeScript, with a focus on clarity, simplicity, and a consistent institutional design.

---

## Development

Install dependencies:

```bash
npm install
````

Run locally:

```bash
npm run dev
```

---

## Build

```bash
npm run build
```

---

## Structure

* `src/pages` — main pages (Home, Learn, Tools, etc.)
* `src/components` — shared UI components
* `src/styles` — global styles and layout system

---

## Notes

The site uses a section-based color system (`band--*`) and avoids visual effects such as gradients, glow, and heavy blur to maintain readability and consistency.

Routing is handled with React Router.
